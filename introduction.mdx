# Camera Plugin for NativePHP Mobile

The Camera Plugin brings powerful media capture capabilities to your NativePHP Mobile applications. Access the device camera, record videos, and let users pick media from their gallery with a clean, event-driven API.

## What You Can Build

With the Camera Plugin, you can:

- **Capture Photos** - Take high-quality photos directly from the device camera
- **Record Videos** - Record videos with customizable duration limits and quality settings
- **Pick from Gallery** - Let users select existing photos and videos from their device
- **Track Operations** - Use optional IDs to correlate captures with specific features
- **Handle Events** - Respond to photo captures, video recordings, and user cancellations
- **Custom Events** - Fire your own event classes for advanced workflows

## Example: Capture a Photo

<CodeGroup>
```php PHP (Livewire/Blade)
use Native\Mobile\Facades\Camera;
use Native\Mobile\Attributes\OnNative;
use Native\Mobile\Events\Camera\PhotoTaken;

class ProfileComponent extends Component
{
    public function takeProfilePicture()
    {
        Camera::getPhoto();
    }
    
    #[OnNative(PhotoTaken::class)]
    public function handlePhotoTaken(string $path)
    {
        // Process the captured photo
        $this->profilePhoto = $path;
    }
}
```

```javascript JavaScript (Vue/React)
import { Camera, On, Off, Events } from '#nativephp';
import { ref, onMounted, onUnmounted } from 'vue';

const photoPath = ref('');

const takePhoto = async () => {
    await Camera.getPhoto();
};

const handlePhotoTaken = (payload) => {
    photoPath.value = payload.path;
};

onMounted(() => {
    On(Events.Camera.PhotoTaken, handlePhotoTaken);
});

onUnmounted(() => {
    Off(Events.Camera.PhotoTaken, handlePhotoTaken);
});
```
</CodeGroup>

## Storage Locations

The plugin automatically manages file storage for you:

**Photos:**
- **Android**: App cache directory at `{cache}/captured.jpg`
- **iOS**: Application Support at `~/Library/Application Support/Photos/captured.jpg`

**Videos:**
- **Android**: App cache directory at `{cache}/video_{timestamp}.mp4`
- **iOS**: Application Support at `~/Library/Application Support/Videos/captured_video_{timestamp}.mp4`

<Note>
All captured media uses temporary storage and is excluded from device backups for optimal performance.
</Note>
